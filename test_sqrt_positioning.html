Circle Formula

Ecuación: (x-h)² + (y-k)² = r

Radio: r = (x-h)² + (y-k)²

Distance Formula

d = (x₁ - x₀)² + (y₁ - y₀)²

La distancia entre dos puntos es: (x₁ - x₀)² + (y₁ - y₀)²

Ellipse Formula

d₁ + d₂ = 2a donde d₁ = (x-c)² + y² y d₂ = (x+c)² + y²

Hyperbola Formula

|d₁ - d₂| = 2a donde d₁ = (x-c)² + y² y d₂ = (x+c)² + y²

Mixed Text Sizes

Small: x² + y² = r²
Normal: x² + y² = r²
Large: x² + y² = r²
XLarge: x² + y² = r²
